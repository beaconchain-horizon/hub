**Horizon Hub** is a community‑driven, low‑fee decentralized marketplace where users can find jobs, sell digital goods, and offer freelance services – built on open‑source blockchain tools and powered by the Horizon ecosystem.

> 🌐 **Live demo:** [https://beaconchain-horizon.github.io/hub](https://beaconchain-horizon.github.io/hub)

---

## ✨ Features

- ✅ **Job board** – post and apply for blockchain, development, design, and security roles
- ✅ **Digital goods marketplace** – sell e‑books, templates, software licenses, API access
- ✅ **Freelance services** – offer consulting, coding, auditing, and training
- ✅ **Real‑time price ticker** – BTC, ETH, Gold, USD (offline‑capable)
- ✅ **Anonymous ad submission** – no mandatory KYC
- ✅ **Local caching** – works offline after first load (IndexedDB / localStorage)
- ✅ **Mobile‑friendly** – responsive design, PWA ready
- ✅ **Monetization built‑in** – commission based on successful transactions
